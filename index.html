AI-Based Internship Recommendation Engine

Project Overview

This project aims to build a simple, lightweight AI-driven recommendation engine to suggest the most relevant internships for candidates applying through the PM Internship Scheme. It targets youth from diverse backgrounds, focusing on ease of use and mobile compatibility.

Problem Statement

With hundreds of internships available, first-generation learners with low digital literacy struggle to find suitable opportunities matching their skills, interests, and locations. This leads to misaligned applications and missed opportunities.

Proposed Solution

  • Capture basic candidate inputs: education, skills, sector interests, and location.
  • Use rule-based or ML-light model to recommend 3-5 top internships.
  • Provide a simple UI with minimal text and visual cues.
  • Support mobile access with regional language adaptability.
  • Output recommendations clearly, using cards or a simple list.

Key Features

  • Lightweight and efficient recommendation algorithm.
  • Intuitive and mobile-first design for broad accessibility.
  • Regional language support for better user engagement.
  • Easy integration with existing PM Internship Scheme portal.
